# Telegram Cloner Bot by Salfetka @calfetkin

Telegram бот для клонирования сообщений между чатами с поддержкой множественных аккаунтов, веб-панели управления и системы зеркал.

## Возможности

- Авторизация аккаунтов по номеру телефона
- Поддержка двухфакторной аутентификации (2FA)
- Сохранение сессий в SQLite базе данных
- Управление аккаунтами через Telegram бота
- Веб-панель управления
- Система зеркал для создания независимых копий бота
- Копирование сообщений из указанного чата в реальном времени
- Отправка сообщений из текстовых файлов
- Управление прокси
- Настройка задержек между сообщениями
- Логирование действий

## Установка

### Требования

- Python 3.8+
- Telegram API ID и API Hash (получить на https://my.telegram.org/apps)
- Токен бота (получить у @BotFather в Telegram)

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Focky1337/cloner_v2
cd salfetka2.0
```

2. Установите зависимости для основного бота:
```bash
pip install -r requirements.txt
```

3. Установите зависимости для веб-панели:
```bash
cd web
pip install -r requirements.txt
cd ..
```

4. Установите зависимости для менеджера зеркал:
```bash
cd mng
pip install -r manager_requirements.txt
cd ..
```

5. Настройте конфигурацию:

Отредактируйте `config.py`:
```python
API_ID = 12345678
API_HASH = "апи хэш"
BOT_TOKEN = "токен бота"
ADMIN_IDS = [айди юзера/клиента]
```

Отредактируйте `mng/manager_config.py`:
```python
MANAGER_BOT_TOKEN = "your_manager_bot_token"
ADMIN_ID = your_telegram_id
CRYPTOBOT_TOKEN = "your_cryptobot_token"
PAYMENT_AMOUNT = 0.0 - цена
```

## Запуск

### Основной бот

```bash
python main.py
```

### Веб-панель

```bash
cd web
python app.py
```

Веб-панель будет доступна по адресу: http://localhost:5000

**Логин для админа:**
- Username: `admin`
- Password: `поставьте свой изменив в файле базы данных`

### Менеджер зеркал

```bash
cd mng
python mirror_manager.py
```

## Использование

### Основной бот

1. Отправьте `/start` боту в Telegram
2. Добавьте аккаунты через меню
3. Настройте источники и назначения для каждого аккаунта
4. Включите режим копирования

### Веб-панель

1. Откройте http://localhost:5000
2. Войдите как администратор или используйте уникальную ссылку для клиента
3. Управляйте аккаунтами, файлами, прокси и настройками через веб-интерфейс

### Менеджер зеркал

1. Отправьте `/start` боту-менеджеру
2. Создайте зеркало, указав токен бота
3. После оплаты (если настроена) зеркало будет активировано
4. Каждое зеркало имеет свою базу данных и изолированное окружение

## Настройки

### Задержки

В `config.py` можно настроить:
- `DELAY_SECONDS` - задержка между сообщениями (по умолчанию 3 секунды)
- `MAX_FILE_SIZE` - максимальный размер файла (по умолчанию 10 МБ)

### Веб-панель

В `web/app.py` можно изменить:
- `app.secret_key` - секретный ключ для сессий Flask
- Логин и пароль администратора (в функции `login()`)
